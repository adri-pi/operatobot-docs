# Bien démarrer avec OperaToBot

# Premières étapes

# Inviter le bot

Première étape essentielle pour utiliser OperaToBot, c'est inviter le bot en cliquant sur le bouton suivant :

Inviter le bot

Vous verrez alors une fenêtre où vous pourrez choisir le serveur sur lequel inviter le bot.

Une fois invité, le bot enverra un message dans l'un des salons (le salon système s'il y en a un), avec les liens essentiels (dont celui vers cette documentation). Ce ne sera pas le cas cependant si vous invitez le bot alors qu'il a déjà été invité sur votre serveur.

Vérifiez bien que le bot que vous invitez est vérifié par Discord sur la page d'invitation de celui-ci ! OperaToBot ne fera jamais de changements sur votre serveur sans votre permission, y compris au moment de son invitation !

# Permissions

Pour que le bot fonctionne correctement, il vous reste un dernier réglage à effectuer : en effet, il vous faut monter le rôle du bot dans la hiérarchie des rôles, afin qu'il puisse effectuer les commandes correctement, notamment pour la modération et la gestion des rôles. En savoir plus

# Premier paramétrage
